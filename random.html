GitGPT

/ AI Agent / Edeucation / Open-Source Analysis tool /

GitGPT is an AI-powered tool designed to assist developers in efficiently comprehending and navigating unfamiliar Github repositories. By following a simple two-step process, this tool enables users to effectively explore and understand new codebases.

First, users can type any link of GitHub repository of their choice. GitGPT will automatically access,examine, and analyze the repository.

Second, when the analysis is complete, users can proceed to the second step, which involves utilizing the AI capabilities of GitGPT to help them find their way around the codebase.

GitGPT is still in experiment. Some features are to be implemented and included in the GitGPT:

  • Automatic generation of repository pathway through analyzing the commit and issue history

  • Dependencies analysis of repository

Mixed Reality visualization of Radioactive Sources

/ Mixed Reality / Particles Detection /

Radiation source localization and characterization are challenging tasks that currently require complex analyses for interpretation. Traditionally, the location of radiation source are displayed in 2D angular space projections on screens. This is unintuitive for users.

To solved this problem, in the summer of 2022, I worked in the University of Michigan Detection for Nuclear Nonproliferation Group(DNNG). Our group proposed propose to display the data directly to the user via the head-mounted MR smart glasses, presenting the directional information as an overlay to the user's 3D visual experience. We describe an open source implementation using efficient data transfer, image calculation, and 3D engine.

AI-Enhanced Interactive Narratives for Programming Education

/ LLM / Programming Educatoin / VSCode Plugin /

LLM is a good tutor for beginners. However, there are still some problems when learning new knowledge with it:

  • The learners may not be clear what questions to ask when they are learning a new field.

  • The learners may not have a concrete learning goal.

  • The learners can't assess their understanding through practice.

To solve these problems, I worked in a group supervised Prof. Steve Oney at University of Michigan to develop the EDBook, an electronic dialogic book embedded with context-aware LLM.

io: An 2D adventure Game - University of Michigan Wolverinesoft

/ Game / Unity / Club /

io is an 2D adventure game that offers challenging gameplay with intricate platforming and intense battles, enhanced by the character's ability to teleport, adding to the player's agility. The game culminates in a demanding boss fight, leaving players in awe of the Crystal Caves' mysterious depths, the Oran Windmill Fort's splendor, and the forlorn atmosphere of the Abandoned City.

It's a group project by the University of Michigan's Wolverinsoft club. I was responsible for the development of the mechanics of the main character and the enemies. We launched this game at University of Michigan's ”Game Show” and garnered 1.2k+ positive reviews within the first week.
